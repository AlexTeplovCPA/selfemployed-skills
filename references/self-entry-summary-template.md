# Self-Entry Summary Template

This is the top-level template for the self-entry output. It defines the overall shape and standard phrasing for any skill generating a grouped-by-form self-entry summary.

For the detailed field-level template, see: `generating-self-entry-summary/references/grouped-by-form-template.md`

---

## Standard opening

Use this opening block at the top of every self-entry summary:

> This summary organizes information by form and field for tax software entry. It does not confirm that every amount is correct, complete, or deductible. Review unresolved items before entering them.

---

## Standard readiness labels

Use these consistently across all field-level status values:

- **Ready for entry** — the information appears organized and supported
- **Clarification required** — one or more items still need follow-up before confident entry
- **CPA review recommended** — material uncertainty exists; a practitioner should review before entry
- **Not applicable** — this field or section does not apply to this file

---

## Standard overall readiness values

Use one of these at the summary level:

- **Complete** — all organized fields appear ready for entry with no material gaps
- **Partial** — some fields are ready, others still need clarification or review
- **Not ready** — the file is not organized enough to support confident entry

---

## Standard stop phrase

Use this phrase when a question is outside what the workflow covers:

> That is outside what this workflow covers. It is a question for a CPA before you file.

---

## Standard closing reminder

Use a variation of this at the end of every self-entry summary:

> This summary does not confirm that every amount is correct, deductible, or ready to file. It organizes available information to support your entry workflow. If unresolved items remain material, consider using the CPA handoff summary before filing.
